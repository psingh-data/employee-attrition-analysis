# Employee Attrition Analysis & Prediction (People Analytics Project)

## Why This Project Matters
Employee attrition is a major challenge for organizations, leading to increased hiring costs, reduced productivity, and loss of valuable talent. This project demonstrates how data-driven insights can help HR teams proactively identify high-risk employees and improve retention strategies.

---

## Project Overview
This project focuses on analyzing employee attrition using data analytics and machine learning techniques. The goal is to identify key factors driving employee turnover and provide actionable insights to improve retention strategies.

---

## Business Problem
Employee attrition leads to increased costs, reduced productivity, and loss of talent. This project aims to answer:

- Why do employees leave?
- Which employees are at high risk?
- How can organizations reduce attrition?

---

## Tools & Technologies
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Machine Learning (Logistic Regression)
- SQL
- Power BI

---

## Data Analysis Approach

### 1. Data Preparation
- Cleaned dataset and handled categorical variables
- Converted attrition into numerical format

### 2. Exploratory Data Analysis (EDA)
- Analyzed attrition trends across salary, age, job role, and satisfaction
- Identified patterns influencing employee turnover

### 3. Machine Learning
- Built Logistic Regression model to predict attrition
- Evaluated model performance

---

## Model Performance
- Model Used: Logistic Regression  
- Accuracy: ~80–85%  
- Successfully identified key drivers of attrition  

---

## Key Insights
- Employees with lower salaries are significantly more likely to leave  
- Overtime is a strong predictor of attrition  
- Job satisfaction plays a critical role in retention  
- Younger employees (25–35) show higher turnover rates
